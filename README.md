# TipJar

Direct tipping and micropayments for creators.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/TipJar.sol`
- Stacks: `contracts/stacks/contracts/tipjar.clar`

## Setup
```bash
pnpm install
pnpm dev
```
